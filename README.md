# 简介
这是一个只有简易控制台界面的打铁，计划支持完整的打铁游戏规则。具体规则请看[打铁规则](Rule.md)

使用方式：控制台启动，其中会显示输入规则。技能代号必须使用全小写。控制台程序会纠正非法输入。

当前模式为测试，需要手动输入人机的技能代号
